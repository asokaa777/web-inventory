# Web Inventory System

Sistem inventory berbasis web yang dikembangkan menggunakan Laravel dan MySQL untuk mengelola data stok barang.

## Fitur

- Login dan Logout
- CRUD Data Barang
- Barang Masuk
- Barang Keluar
- Manajemen Stok
- Database MySQL

## Teknologi

- Laravel
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Tujuan Project

Project ini dibuat untuk membantu pengelolaan inventaris barang secara digital sehingga proses pencatatan stok menjadi lebih efektif dan terorganisir.
